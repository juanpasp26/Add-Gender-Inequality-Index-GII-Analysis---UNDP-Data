# ♀️ Gender Inequality & Human Development Analysis

### 🔍 Project Overview
This project examines how gender equality is linked to a country's overall development. Using data from the **United Nations Development Programme (UNDP)**, I analyzed the Gender Inequality Index (GII) and its components to identify global patterns in education, labor, and health.

### 📊 Research Questions
1. **How does HDI affect equality?** Investigating the link between human development rankings and gender-based disparities.
2. **Component Analysis:** Evaluating how factors like maternal mortality, adolescent birth rates, and labor force participation contribute to the overall index.

### 🛠️ Key Technical Skills
* **Data Preprocessing:** Handled a complex Excel dataset by converting character strings to numeric types and managing missing values (`NA`).
* **Correlation Analysis:** Used Pearson correlation to identify strong links (e.g., the **0.90** correlation between GII and HDI rank).
* **Statistical Visualization:** Created boxplots and scatter plots to identify outliers and trends in global inequality.

### 📈 Key Insights
* **HDI Impact:** There is a very strong correlation (**0.90**) showing that higher human development levels significantly increase the chances of gender equality.
* **Education Parity:** Found a near-perfect correlation (**0.97**) between male and female secondary education, suggesting a global trend toward educational equality.
* **Labor Gap:** Discovered a weak correlation (**0.46**) in labor force participation, highlighting that significant inequality still exists in the global workforce compared to education.
